About

sOrg is a collection of free and open-source software. It provides both simple graphical and advanced CLI tools. Most of the software is written in C and C++ programming languages.

Tools

sCore - A collection of CLI tools for basic system management and info.
sMarkup - A CLI markup language similar to Markdown. Can be used for writing docs.
sWelcomer - A graphical customizable startup welcomer made with SDL2
